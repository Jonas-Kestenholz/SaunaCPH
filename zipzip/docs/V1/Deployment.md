# Deployment

## Build strategi

Appen bygges via Expo EAS.

---

## Workflow

1. Udvikling lokalt
2. Commit til repo
3. Build via EAS
4. Upload til TestFlight
5. Feedback og iteration

---

## CI/CD (fremtidigt)

Planlagt via Azure DevOps:

* Automatisk build ved push
* Test kørsel
* Deployment pipeline

---

## Apple setup

* Apple Developer konto kræves
* App uploades via App Store Connect

---

## Fordele

* Automatiseret proces
* Ensartede builds
